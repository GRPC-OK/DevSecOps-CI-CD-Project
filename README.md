## 프로젝트 구조

```
├── app/                # App Router 기반 페이지 및 컴포넌트
├── public/            # 정적 파일 (이미지, 폰트 등)
├── styles/            # CSS 스타일시트
└── components/        # 재사용 가능한 컴포넌트
```

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```
